Connexion Google Sheets

Exportez vos données vers Google Sheets pour les sauvegarder dans le cloud, y accéder depuis votre téléphone, ou les partager avec votre comptable.

📋 Guide de configuration — À faire une seule fois DÉBUTANT
1

Créer votre Google Sheet

Allez sur sheets.google.com → Créer un nouveau fichier → Nommez-le "DAR SADIK".
Créez 5 onglets (feuilles) avec ces noms exacts :

Ventes Gasoil Paiements Clients Camions

Pour créer un onglet : cliquez sur le "+" en bas à gauche de la page Sheets.

2

Ouvrir Apps Script

Dans votre Google Sheet → Menu Extensions → Apps Script. Une nouvelle page s'ouvre.

3

Coller le code Apps Script

Supprimez tout le texte dans l'éditeur, puis copiez-collez ce code :

function doPost(e) {
  try {
    var data = JSON.parse(e.postData.contents);
    var sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(data.sheet);
    if (!sheet) return ContentService.createTextOutput(JSON.stringify({error:'Sheet not found'})).setMimeType(ContentService.MimeType.JSON);
    
    if (data.action === 'export') {
      sheet.clearContents();
      if (data.rows && data.rows.length > 0) {
        sheet.getRange(1, 1, data.rows.length, data.rows[0].length).setValues(data.rows);
        sheet.getRange(1, 1, 1, data.rows[0].length).setFontWeight('bold').setBackground('#0d3d6e').setFontColor('#ffffff');
      }
      return ContentService.createTextOutput(JSON.stringify({ok:true, written: data.rows ? data.rows.length-1 : 0})).setMimeType(ContentService.MimeType.JSON);
    }
    
    if (data.action === 'import') {
      var values = sheet.getDataRange().getValues();
      return ContentService.createTextOutput(JSON.stringify({ok:true, rows: values})).setMimeType(ContentService.MimeType.JSON);
    }
    
    return ContentService.createTextOutput(JSON.stringify({error:'Unknown action'})).setMimeType(ContentService.MimeType.JSON);
  } catch(err) {
    return ContentService.createTextOutput(JSON.stringify({error: err.message})).setMimeType(ContentService.MimeType.JSON);
  }
}

function doGet(e) {
  return ContentService.createTextOutput('DAR SADIK API OK').setMimeType(ContentService.MimeType.TEXT);
}
4

Déployer comme application web

Dans Apps Script → bouton Déployer → Nouveau déploiement → Type : Application Web
Paramètres importants :
• Exécuter en tant que : Moi
• Qui a accès : Tout le monde
→ Cliquez Déployer → Autorisez l'accès → Copiez l'URL de déploiement

⚠️ L'URL ressemble à : https://script.google.com/macros/s/ABC.../exec
